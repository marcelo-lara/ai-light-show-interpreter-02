# Quickstart: Audio to DMX Generator

## Prerequisites
- Docker & Docker Compose
- Ensure input directory `data/songs/` has `.mp3` files.
- Ensure audio artifacts (like `data/essentia/fft_bands.json`) are already generated by upstream pipelines.

## Execution

1. Build the Docker environment:
   ```bash
   docker compose build
   ```
2. Start the interactive CLI:
   ```bash
   docker compose run --rm light-show-cli
   ```
3. Use Arrow Keys to navigate the playlist of `.mp3` tracks (Select `Cinderella - Ella Lee` for baseline testing).
4. Press `Enter` to select a track.
5. The `Evaluator` will parse the `stage_virtual_canvas.json`, extract 2D fixture coordinates, process the 5-band FFT `q_buffer`, inject it into the `Radial Pulse` and `Linear Wave` shaders, and bake a binary 50FPS `.dmx` sequence into `data/shows/`.

## Verification

Check the output directory for your compiled show:
```bash
ls -la data/shows/*.dmx
```
